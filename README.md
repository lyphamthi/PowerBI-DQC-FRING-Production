# Manufacturing Quality Dashboard

## Overview
Dashboard theo dõi sản lượng và tỷ lệ lỗi trong dây chuyền sản xuất dòng sản phẩm UDISC và FRING.

## Tools
- Power BI
- Power Query
- DAX

## KPIs

### Production Monitoring
- Sản lượng theo công đoạn
- Sản lượng theo ngày
- Theo dõi hiệu suất sản xuất
- Top 10 tỉ lệ lỗi nhiều nhất xếp theo tên 

### Quality Monitoring
- Tỷ lệ lỗi tổng
- Tỷ lệ lỗi theo công đoạn
- Phân tích chi tiết lỗi
